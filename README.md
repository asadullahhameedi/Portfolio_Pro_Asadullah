## Features

- **Animated Loading Screen** — Pulsing logo with progress bar
- **Particle Canvas Background** — Interactive floating particles that respond to mouse movement
- **Scroll Progress Bar** — Gradient progress indicator at the top of the page
- **Typing Effect** — Dynamic role titles typed out in the hero section
- **Glassmorphism UI** — Frosted glass cards with backdrop blur throughout the design
- **Animated Skill Bars** — Progress bars that animate into view on scroll
- **Counter Animation** — Stats (20+ Projects, 15 Technologies, 3+ Years, 10+ Clients) animate when visible
- **Scroll-triggered Animations** — Fade-in-up, fade-in-left, fade-in-right, and scale-in transitions
- **Responsive Navigation** — Desktop navbar with hover underlines + full-screen mobile menu
- **Project Cards** — Hover overlays with Live Demo and GitHub links
- **Experience Timeline** — Alternating timeline layout for career and education history
- **Contact Form** — Validated form with toast notifications
- **Back to Top Button** — Appears on scroll with smooth animation
- **Custom Scrollbar** — Gradient-styled scrollbar matching the theme
- **Blob Morphing** — Organic animated background shapes
- **Orbital Rings** — Rotating decorative circles in the hero section
- **Fully Responsive** — Optimized for mobile, tablet, and desktop

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Asad321788/your-repo-name.git
   ```

2. Navigate to the project directory:
   ```bash
   cd your-repo-name
   ```

3. Open the HTML file in your browser:
   ```bash
   open Asadullah-portfolia.html
   ```
   Or simply double-click the file in your file explorer.

> **No npm install, no build step, no server needed** — it just works out of the box.

## Customization

Want to use this as your own portfolio template? Here's what to change:

| What | Where |
|------|-------|
| Name & Title | Search for `Asadullah Hameedi` and `Full-Stack Developer` |
| Typing Texts | Find the `typingTexts` array in the `<script>` section |
| Skill Percentages | Update `--skill-width` CSS variables and `data-skill` attributes |
| Projects | Replace project cards with your own titles, descriptions, links, and images |
| Contact Info | Update email, phone, WhatsApp, and social links in the Contact section and Footer |
| Colors | Modify the `tailwind.config` custom color palette and CSS gradient values |
| Profile Image | Replace the placeholder `AH` avatar div with an `<img>` tag pointing to your photo |
